# MAIRIE SHOP

نسخة React احترافية من موقع MAIRIE SHOP (RTL).

التشغيل محلياً:
1. تأكد أن لديك Node.js مثبت.
2. انسخ الملفات في مجلّد مشروع جديد (أقترح Vite).
3. نفّذ:
   ```bash
   npm install
   npm run start
   ```
4. افتح http://localhost:5173

تحسينات مقترحة لاحقاً:
- تحويل إلى TypeScript.
- رفع الصور إلى CDN أو استخدام خدمة صور محسّنة.
- إضافة صفحة إدارة منتجات أو ربط API.
- سلة مشتريات وCheckout.
- اختبارات واجهة مستخدم وUnit tests.
